# Heritage 4.0 — Paper Supplementary Materials

This is the entry point for supplementary materials referenced from the Heritage 4.0 paper (Springer CCIS, 6 pages).

The full evidence base — codebook, transcripts, analytical runs, narrative reports — lives in:

→ **[InSites-Brain/Heritage4.0/research-tools/](../InSites-Brain/Heritage4.0/research-tools/)**

That folder is organized into three subfolders by reading priority:

- **[core/](../InSites-Brain/Heritage4.0/research-tools/core/)** — Files cited directly in the paper. Start here.
- **[reproducibility/](../InSites-Brain/Heritage4.0/research-tools/reproducibility/)** — Three independent extraction runs validating the central findings.
- **[further-reading/](../InSites-Brain/Heritage4.0/research-tools/further-reading/)** — Narrative analyses with source-level tracing.

A reader-oriented guide explaining the methodology, the case study, and the rationale for the paper's headline figures (including why the paper reports 80% rather than 94%) is in the main supplementary README:

→ **[research-tools/README.md](../InSites-Brain/Heritage4.0/research-tools/README.md)**
