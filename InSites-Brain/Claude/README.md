# אתר.בוט — סדנת בצלאל (Claude.ai)

אתר.בוט הוא Claude Project לסיוע בהערכת משמעות תרבותית של חפצים ויצירות, לפי שיטת **CBSA** (Context-Based Significance Assessment) — בעברית, באמצעות התכתבות מובנית בשבעה שלבים (0–6).

## 🔗 קובץ ההנחיות לפרויקט [InSites-CAA-claude.md](InSites-CAA-claude.md) — קובץ ההנחיות המלא של הבוט 

לחצו על הכפתור **Copy raw file** בפינה הימנית-עליונה של תצוגת GitHub → הדביקו בשדה Project Instructions.

## כיצד להגדיר את הפרויקט ב-Claude.ai

1. היכנסו ל-[claude.ai](https://claude.ai) → **Projects** → **Create Project**
2. תנו לפרויקט את השם `אתר.בוט`
3. גשו לשדה **Project Instructions** והדביקו את תוכן `InSites-CAA-claude.md`
4. בצ'אט חדש בתוך הפרויקט: העלו את המסמך שברצונכם להעריך והקלידו `התחל`

## המלצות

- חשבון **Claude Pro** ומעלה
- דגם **Opus** עם **Extended Thinking** פעיל

## תמיכה בעברית

הבוט עוקב אחר שפת ההוראה: כתבו בעברית — הפלט יהיה בעברית, כולל הדאשבורד וגרף הידע (KG).
