Analysis: What to Translate

  Recommendation: Translate everything — UI + content. The site is a companion to an English-language CAA
  workshop. Participants need to understand all content — stage explanations, tool descriptions, research query   
  prompts, etc. Leaving Hebrew content behind a translated UI would create a confusing bilingual experience.      

  Exception: The sample heritage texts (DEMO_DATA, ZAIRA_TEXT, AYELET_WT_TEXT) are Hebrew heritage data used for  
  the KG demo. These can stay as-is (they're domain data, not UI), but should be clearly labeled as Hebrew        
  examples, or ideally replaced with English heritage examples.

  ---
  תכנית עבודה — המרת האתר המלווה לאנגלית + הוספת תכנית סדנה

  שלב 1: הכנת תשתית — RTL → LTR

  - index.html: שינוי lang="he" dir="rtl" → lang="en" dir="ltr", עדכון <title>
  - App.tsx: שינוי dir="rtl" הגלובלי ל-dir="ltr" (מופיע ב-~8 מקומות)
  - כל הקומפוננטות עם dir="rtl" מפורש: AboutView, StepsList, StepDetailView, MobileNav, Sidebar, WelcomeOverlay,  
  footer
  - החלפת text-right → text-left במקומות רלוונטיים
  - ChevronLeft → logic flip (הכיוונים מתהפכים ב-LTR)
  - Font: Assistant (Hebrew-optimized) → פונט אנגלי מתאים (למשל Inter) או השארת Assistant (תומך גם באנגלית)       

  שלב 2: תרגום הממשק (UI Labels)

  קבצים מושפעים:
  - Header.tsx: כותרת "סדנת אתר.בוט" → "Atar.Bot Workshop", tooltips
  - MobileNav.tsx: טאבים "בית/שלבים/הרחבות/מה באתר" → "Home/Stages/Tools/About"
  - Sidebar.tsx: כותרת, כפתור "הרחבות וכלים", tooltip
  - AboutView.tsx: תיאור הפרויקט, כפתורי ניווט
  - WelcomeOverlay.tsx: כותרת המודל
  - StepsList.tsx: breadcrumb, כותרת
  - StepDetailView.tsx: כותרות סקשנים ("למה שלב זה חשוב?", "מה קורה?", "ההנחיות לבוט"), placeholders, כפתורים     
  - App.tsx: breadcrumbs, home view (ResourceGroups, labels, footer), tools view (כותרות), error messages

  שלב 3: תרגום התוכן (constants.tsx)

  הקובץ הגדול ביותר — כל התוכן המקצועי:
  - CORE_AGENTS (7 items): שמות ותפקידי השלבים → אנגלית
  - SUPPORT_AGENTS (5 items): שמות כלים → אנגלית
  - STEP_DETAILS (7 stages × 4 fields): whyImportant, whatHappens, cognitiveLink, promptSummary, extensions →     
  אנגלית
  - PROMPT_TRANSLATIONS (stages 0-6): Hebrew prompts → English (note: PROMPT_PREVIEWS_EN already exists in        
  English!)
  - DIALOGUE_PRINCIPLES: 3 principles → English
  - PROMPT_ADVISOR_SYSTEM: system prompt → English
  - RESEARCH_QUERIES: titles, descriptions, full prompts (5 items + sub-queries) → English
  - EDUCATIONAL_PROMPT: → English
  - MARC_INSTRUCTIONS: כבר קיים באנגלית! רק להפוך את en לברירת מחדל
  - GRAPH_PROMPT: כבר באנגלית חלקית, לעדכן
