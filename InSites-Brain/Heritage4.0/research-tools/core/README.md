# Core — paper-cited evidence

These files are referenced directly by the Heritage 4.0 paper. If you want to verify a methodological claim or inspect the analyzed data, start here.

- **[SKILL.md](SKILL.md)** — Claim-extractor codebook v1.1. Defines what counts as a claim and the classification scheme (epistemic level, expert response, content type).
- **[Claim-Level-Count-2303.md](Claim-Level-Count-2303.md)** — The 45-claim manual count cited in §4. First-pass paragraph-level extraction by single analyst.
- **[claim-extraction-cross-run-comparison.md](claim-extraction-cross-run-comparison.md)** — Three-run convergence analysis. Basis for the 80% synthesis/inference figure in the paper.
- **[tz-textonly.md](tz-textonly.md)** — Raw session transcript (chat export). The full conversation as it happened.
- **[tuba-session-transcript.md](tuba-session-transcript.md)** — Same conversation, cleaned and organized by CBSA stage. Adds a Session Report section not in the raw export.
